# Brand Guidelines

> Customize this file with your company's brand identity. Skills that generate HTML reports
> (serp-clusters, content-auditor, content-brief) will use these values for styling.

## Quick Setup

1. Replace `logo-light.svg` and `logo-dark.svg` in this directory with your logos
2. Edit `brand-config.json` with your colors, fonts, and company info
3. Reports will automatically pick up your branding

## Color Palette

### Primary Colors
| Name | Hex | Usage |
|------|-----|-------|
| Primary | `#3B82F6` | Main accent, CTAs, links |
| Primary Light | `#60A5FA` | Hover states, highlights |
| Secondary | `#10B981` | Success indicators, secondary accent |

### Background Colors
| Name | Hex | Usage |
|------|-----|-------|
| Background | `#FFFFFF` | Main page background |
| Background Dark | `#0F172A` | Dark mode / header background |
| Background Card | `#F8FAFC` | Card backgrounds, table alternating rows |

### Text Colors
| Name | Hex | Usage |
|------|-----|-------|
| Text Primary | `#1E293B` | Headlines, important text |
| Text Secondary | `#64748B` | Body text |
| Text Muted | `#94A3B8` | Captions, footnotes |

### Status Colors
| Name | Hex | Usage |
|------|-----|-------|
| Success | `#22C55E` | Positive changes, improvements |
| Warning | `#F59E0B` | Attention needed |
| Error | `#EF4444` | Negative changes, issues |

### Gradients
```css
/* Primary brand gradient */
background: linear-gradient(135deg, #3B82F6 0%, #60A5FA 100%);

/* Header gradient */
background: linear-gradient(90deg, #0F172A 0%, #1E293B 100%);
```

## Typography

### Font Family
- **Primary**: Inter (weights 400-700)
- **Mono**: JetBrains Mono (for code/data)
- **Fallback**: system-ui, -apple-system, sans-serif

### Font Sizes
| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| Report Title | 28-32px | 700 | 1.2 |
| Section Heading | 20-24px | 600 | 1.3 |
| Subsection | 16-18px | 600 | 1.4 |
| Body | 14-16px | 400 | 1.6 |
| Caption | 12-13px | 400 | 1.4 |
| Data/Numbers | 14px | 500 (mono) | 1.4 |

## Logo

Place your logo files in this directory:
- `logo-light.svg` — for light backgrounds (dark logo)
- `logo-dark.svg` — for dark backgrounds (light logo)

Recommended: SVG format, max 200px wide, transparent background.

If no logo files are present, reports will display your company name from `brand-config.json` as text.
