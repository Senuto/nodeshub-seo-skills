# Branding Assets

This directory contains your brand identity for HTML reports generated by SEO skills.

## Files

| File | Purpose |
|------|---------|
| `brand-config.json` | Colors, fonts, company info — read by report generators |
| `brand-guidelines.md` | Human-readable brand reference for Claude |
| `logo-light.svg` | Logo for light backgrounds (replace with yours) |
| `logo-dark.svg` | Logo for dark backgrounds (replace with yours) |
| `extract-brand-styles.js` | Browser DevTools script to extract styles from any website |

## Quick Setup

### Option 1: Manual
1. Replace `logo-light.svg` and `logo-dark.svg` with your logos
2. Edit `brand-config.json` — update company name, colors, fonts

### Option 2: Extract from your website
1. Open your website in a browser
2. Open DevTools (F12) → Console tab
3. Paste the contents of `extract-brand-styles.js` and press Enter
4. Copy the `brand-config.json` output into this directory

## How it works

Python scripts read `brand-config.json` at report generation time:

```python
from branding import load_brand
brand = load_brand()  # returns dict from brand-config.json
# brand['colors']['primary'] → '#3B82F6'
# brand['company']['name'] → 'Your Company'
```

If `brand-config.json` is missing or unreadable, reports fall back to default Nodeshub styling.
